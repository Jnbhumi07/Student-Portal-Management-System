# 🎓 Student Portal Management System

## 📌 Project Objective

The Student Portal Management System is a web-based application that allows students to access their academic information online.

Students can:
- Login
- View Attendance
- View Marks
- View Assignments
- View Notices
- Update Profile

Admin can:
- Add Student
- Update Student
- Delete Student
- Manage Attendance
- Manage Marks
- Upload Notices

## 💻 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Java

Database:
- MySQL
