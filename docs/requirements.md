# 📋 Student Portal Management System - Requirements

## 🎯 Project Objective

The Student Portal Management System is a Java-based application designed to manage student information efficiently. The system will allow students to view their academic details and enable administrators to manage student records.

---

# 👨‍🎓 Student Features

- Student Login
- View Profile
- View Attendance
- View Marks
- View Notices
- View Assignments
- Update Profile
- Logout

---

# 👨‍💼 Admin Features

- Admin Login
- Add Student
- View Student Details
- Update Student Details
- Delete Student
- Manage Attendance
- Manage Marks
- Upload Notices
- Search Student
- Logout

---
